# steam-apps

This repository contains Rust scripts and helper functions for collecting Steam AppIDs.

## Output URLs

- `https://carapacik.github.io/steam-apps/apps.all.json`
- `https://carapacik.github.io/steam-apps/apps.games.json`
- `https://carapacik.github.io/steam-apps/appids.all.json`
- `https://carapacik.github.io/steam-apps/manifest.json`
